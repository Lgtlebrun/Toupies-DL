=============================== TESTS ======================================

Vous vous trouvez dans le répertoire contenant les fichiers source des classes "test unitaire".

Vous trouverez ici un main dans "testAll.cpp" permettant de tester toutes les classes à la fois.


Pour le construire, un Makefile est à votre disposition. Executez "make" dans votre terminal.
Un executable nommé "TestAll" sera alors construit.


Lorsqu'un test échoue, un rapport d'erreur est écrit au format "Test[classe défaillante].txt".

TestIntegrateur produit des fichiers textes contenant des données sur la Bille et l'oscillateur.
Les deux fichiers octave à votre disposition permettent de les plotter.

Les fichiers StatsConiqueNewmark et StatsConiqueRK4 permettent la comparaison de ces deux
intégrateurs sur une simple toupie conique.


TestAll est une application console, à executer dans un terminal.
