============================ README - Ultimate Toupie Simulator ==================================

Dans ce dossier se trouve le code de l'édition grand public de notre simulateur.

L'interface graphique minimaliste du launcher permet le lancement rapide d'une simulation
ne comportant qu'un unique objet, aux caractéristiques précisées avec... précision par 
l'utilisateur.

Cette édition a également la spécificité de permettre des simulations multiples orchestrées
par des intégrateurs différents.



MAIN : UltimateMain.cpp

CLASSES : LauncherAccueil, BordMode, Grillage, FichierSearch, Bloc2Boutons, BoutonPosition, TypeBouton,
	  QTextViewer, TextEdit + les classes du répertoire general

POLICES : Copper-Black, Agency FB

IMAGES : Bille, icone, toupie-bleue-marine, tav, cone, chinoise, oscillateur, oeil, BoutonBleu, pastilleGrise



Installation : QMake, puis make dans ce dossier.
